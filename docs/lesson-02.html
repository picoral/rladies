Reading multiple files in R

For this tutorial on how to read data in R, we will be using bilingual children production from the (Corpus of bilingual childrenâ€™s speech on kaggle)[https://www.kaggle.com/datasets/rtatman/corpus-of-bilingual-childrens-speech].

There are multiple files to be downloaded. Make sure you know where the files are in your project.

Getting started

As usual we will start by loading tidyverse.

library(tidyverse)

Then we will list all the files we want to read in,

files2read <- list.files(path = "corpus-data/childrens-speech",
                         pattern = "\\.cha",
                         full.names = TRUE)

Then we will use read_csv to read all the files at once, setting the id to be our filename variable.

data <- read_delim(files2read,
                   col_names = FALSE,
                   delim = "âˆ‘",
                   id = "filename")
data <- data %>% 
  mutate(participant = str_remove_all(filename, "corpus-data/childrens-speech/|\\.cha"))
participant_info <- data %>% 
  filter(str_sub(X1, 1, 1) == "@")
data <- data %>% 
  filter(str_sub(X1, 1, 1) != "@")
data <- data %>% 
  mutate(speaker = str_sub(X1, 1, 5),
         speaker = if_else(str_sub(speaker, 1, 1) %in% c("*", "%"),
                             speaker, NA_character_)) %>% 
  fill(speaker)
data %>% 
  count(speaker)
data <- data %>% 
  mutate(utterance = str_remove(X1, fixed(speaker)))
data %>% 
  filter(speaker == "*CHI:") %>% 
  count(participant)
